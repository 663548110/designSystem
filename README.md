# Design System Rules Templates

这个仓库用于维护一套可被 Figma 端、代码端和工具端共同消费的设计系统规则模板。

## 目录结构

- `rules.md`
  - 全局规则
  - 权威顺序
  - Figma 端与代码端的共享工作流
- `tokens.md`
  - 设计 token 真相
  - token 映射来源
- `components/README.md`
  - 组件条目导航
  - 维护约定
- `components/_template.md`
  - 单个组件条目的模板

## 维护原则

1. 单一数据源
2. 组件详情一组件一文件
3. token 与组件详情分离
4. 工具优先解析结构化 YAML 代码块
