# Components Index

这里维护组件条目的导航，不重复维护组件详情字段。

## 维护约定

1. 一组件一文件
2. 文件名优先与组件库一级目录保持一致
3. 组件详情只写在对应文件，不在本文件重复
4. 工具读取时，以 `components/*.md` 为唯一组件数据源
5. 本文件只做导航，不作为组件事实来源
6. 若需要组件清单，应由工具扫描 `components/*.md` 生成
